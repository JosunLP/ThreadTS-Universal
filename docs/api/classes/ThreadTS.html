    Class ThreadTS

    ThreadTS - The main class for parallel computing.

    Implements a singleton pattern for resource sharing while allowing configuration customization. Provides methods for executing functions in parallel, including map, filter, reduce, and batch operations.

    Example

    // Get the singleton instance
    const instance = ThreadTS.getInstance();

    // Execute a function in parallel
    const result = await instance.run((x) => x * 2, 21);

    // Process arrays in parallel
    const doubled = await instance.map([1, 2, 3], (x) => x * 2);
