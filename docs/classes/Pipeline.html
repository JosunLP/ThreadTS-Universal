    Class Pipeline<T>

    Pipeline class for fluent chaining of parallel operations. Supports lazy evaluation - operations are only executed when execute() is called.

    Example

    const result = await ThreadTS.pipe([1, 2, 3, 4, 5])
      .map(x => x * 2)
      .filter(x => x > 4)
      .reduce((acc, x) => acc + x, 0)
      .execute();

    Type Parameters

    • T

      The current type of array elements
